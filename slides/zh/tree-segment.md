## 線段樹

線段樹是一種二元樹結構，把陣列的每個區間對應到一個節點，讓區間查詢與區間更新都能在 $O(\log n)$ 內完成。

---

## 核心概念

每個節點負責一個區間 $[lo, hi]$ 並存放該區間的聚合值（此處為區間和）；葉節點對應單一元素，內部節點的值由兩個子節點合併而成。

- 區間查詢：從根下降，完全被覆蓋的節點直接取值，不相交則略過。
- lazy propagation：區間更新時在被覆蓋的節點掛上「lazy 標記」，延後傳給子節點。
- lazy 標記在之後查詢或更新下降經過該節點時才被推下去。

---

## 運作流程

1. 區間更新 $[ql, qr]$：完全被覆蓋的節點掛 lazy 標記並更新其聚合值。
2. 部分相交的節點先把自己的 lazy 標記推給子節點，再遞迴。
3. 區間查詢時同樣下降，經過帶 lazy 標記的節點就推下標記，確保讀到的值正確。

## 複雜度分析

| 操作 | 複雜度 |
| --- | --- |
| 建樹 | $O(n)$ |
| 區間查詢 | $O(\log n)$ |
| 區間更新（lazy） | $O(\log n)$ |
| 空間 | $O(n)$ |

$$T_{\text{query}}(n) = O(\log n)$$

查詢路徑在每一層最多展開常數個節點，故與樹高 $\log n$ 成正比。

## 程式碼

```cpp
void update(int node, int lo, int hi, int ql, int qr, long long val) {
    if (qr < lo || hi < ql) return;
    if (ql <= lo && hi <= qr) { applyLazy(node, lo, hi, val); return; }
    pushDown(node, lo, hi);
    int mid = (lo + hi) / 2;
    update(2 * node, lo, mid, ql, qr, val);
    update(2 * node + 1, mid + 1, hi, ql, qr, val);
    tree[node] = tree[2 * node] + tree[2 * node + 1];
}
```

## 優缺點與使用時機

- 優點：同時支援區間查詢與區間更新，皆為 $O(\log n)$。
- 優點：聚合函式可替換（和、最小值、最大值、GCD 等）。
- 缺點：實作較 Fenwick 樹複雜，常數較大。
- 適用：需要區間更新、或需要非加法聚合的場景。

---

## 小結

- 把陣列區間組成二元樹，查詢與更新沿樹高進行。
- lazy propagation 把區間更新延後到真正需要時才下推。
- 區間查詢與區間更新皆為 $O(\log n)$。
